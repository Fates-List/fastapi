It creates the following Form request parameters in your endpoint:

grant_type: the OAuth2 spec says it is required and MUST be the fixed string “password”.

Nevertheless, this dependency class is permissive and allows not passing it. If you want to enforce it, use instead the OAuth2PasswordRequestFormStrict dependency.
